Note to distributors:

Please note and retain the serial number of the new LCD and the matching pantone profile name in case

the motherboard requires replacing at a later date.

 

1.      If not already installed, install the Control Center application from the Device Drivers & Utilities + User's Manual disc.

2.      Note your LCD panel's serial number

3.      You will now need to download the the RwI2CTool and the appropriate pantone profile file.

• When prompted "Enter the LCD or System Serial Number."

• Click on Download Tool or Download Profile (you will need to download both)

4.      The RwI2CTool and/or Pantone profile file will now download.

5.      Copy the unzipped RwI2CTool to a location on your computer (this folder will contain the appropriate utilities for writing the

profile to the EEPROM on your computer).

6.      Copy the Pantone profile file for your display into the RwI2CTool folder (it is important that the profile is located inside the

tool folder as the tool will look for the profile in this location).

7.      Go to the Start menu in Windows and run the Command Prompt (the Command Prompt is located in the Windows System

folder in the Start menu).

8.      Within the Command Prompt you will then need to navigate to the folder path with the RwI2CTool, and then run the

RwI2CTool.exe followed by the Pantone profile filename e.g. RwI2CTool_xxxxxxxxxxx

9.      After the file has executed you will need to run the X-Rite Color Assistant app to Restore profiles in Settings.
